# System Log

The System Log card displays real-time log entries from the PSWebHost server.

## Features

- **Real-time Updates**: Log entries stream automatically as they occur
- **Severity Filtering**: Filter by log severity (Info, Warning, Error, Debug)
- **Search**: Full-text search across all log fields
- **Category Filtering**: Filter by log category

## Log Severity Levels

| Level | Description |
|-------|-------------|
| **Info** | Normal operational messages |
| **Warning** | Potential issues that may need attention |
| **Error** | Error conditions that require investigation |
| **Debug** | Detailed debugging information |

## Usage

1. Open the System Log card from the main menu
2. Use the filter controls at the top to narrow down entries
3. Click on a log entry to see full details
4. Use the refresh button to force a log refresh

## Configuration

Log retention and output settings can be configured in the server's `config/settings.json` file.
