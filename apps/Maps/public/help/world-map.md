# World Map

The World Map card displays geographic data and location-based information.

## Features

- **Interactive Map**: Pan, zoom, and interact with the map
- **Location Markers**: Display points of interest or data locations
- **Region Highlighting**: Highlight countries or regions
- **Data Overlays**: Visualize data geographically

## Usage

1. Open the World Map from the main menu
2. Use mouse wheel or pinch gestures to zoom
3. Click and drag to pan the map
4. Click on markers for detailed information

## Data Sources

The map can display data from various sources:
- Server locations
- User activity by region
- Geographic metrics and statistics

## Configuration

Map appearance and data sources can be customized through the card settings.
